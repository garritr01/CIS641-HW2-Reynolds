Term Project Summary - Website

VAR Group's goal is the creation a system that allows the user to schedule, record, and analyze data from their life easily and in one place.

Scheduling - The user can schedule events on specific days, or recurring daily, weekly, monthly, or annually and be reminded an hour ahead of time if desired.

Recording - The user can design an interface specific to the event for specific data collection of repeated events, or just notes.

Analysis - The user can visualize time usage and filter events by the data they've collected and compare other data points based on this filtration.

This project will both produce a personally beneficial system, and allow me to gain invaluable experience. Designing software for deployment in a team is a new experience for me that I expect to gain valuable programming and project management experience from.
